Система управления автосалоном

Описание

Данное приложение представляет собой консольную утилиту для управления автосалоном. Программа позволяет работать с каталогом автомобилей, продажами и расходами, используя данные, хранящиеся в памяти.

Предметная область

В рамках приложения рассматриваются три основные предметные области:

Автомобили

Система хранит и обрабатывает информацию об автомобилях:
идентификатор,
SKU (уникальный код),
название,
категория,
цена,
количество на складе.

Также реализована работа с категориями автомобилей.

Продажи

Система позволяет управлять продажами:
создание продажи,
добавление товаров в продажу,
указание количества и цены,
расчёт общей суммы продажи,
изменение статуса продажи,
получение продаж по статусу,
расчёт выручки по статусу,
поиск продажи по идентификатору.

Расходы

Система учитывает расходы:
дата операции,
сумма,
категория расхода,
описание.

Функциональность

Работа с автомобилями:
добавление категорий,
добавление автомобилей,
поиск автомобилей по названию,
получение списка автомобилей, отсортированных по цене,
получение автомобилей по категории.

Работа с продажами:
создание продажи,
добавление позиций в продажу,
изменение статуса продажи,
получение продаж по статусу,
расчёт общей выручки по статусу,
получение продажи по идентификатору.

Работа с расходами:
добавление категорий расходов,
добавление расходов,
расчёт общей суммы расходов за период,
получение расходов по категориям,
получение последних N расходов.

Архитектура

Приложение реализовано с использованием принципов SOLID:
разделение на слои Entities, Repositories, Services и Console.
использование интерфейсов для взаимодействия между слоями.
внедрение зависимостей через конструкторы.

Хранение данных

Данные хранятся в оперативной памяти с использованием коллекций List и Dictionary.
